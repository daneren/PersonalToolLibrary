# PersonalToolLibrary

## 快捷命令

### Vscode远程连接docker容器配置

```shell
wget https://raw.githubusercontent.com/daneren/PersonalToolLibrary/refs/heads/main/hf_tools/docker_tools/ssh2docker.sh && bash ssh2docker.sh
```

### 快速添加定义命令
```shell
wget https://raw.githubusercontent.com/daneren/PersonalToolLibrary/refs/heads/main/costum_commands.sh && cat costum_commands.sh >> ~/.bashrc && source ~/.bashrc
```
